# Wikipedia Search App

A Next.js application that allows users to search for Wikipedia articles and displays the results as links to their respective Wikipedia pages. This project is built with React, TypeScript, and is deployed on Vercel.

## Features

- Search for articles using the Wikipedia API.
- Display search results with links to Wikipedia pages.
- Fast and responsive UI powered by Next.js.
- Built with TypeScript for type safety and better development experience.

## Tech Stack

- **Next.js**: Framework for server-side rendering and static site generation.
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript providing static typing.
- **Vercel**: Deployment platform for modern web applications.
